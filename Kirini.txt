## 🔧 관리자 Action
(관리자가 페이지별로 할 수 있는 역할 및 기능 정리)

### 정책 초안
- 콘텐츠는 게시판에서 직접 관리
- 관리 내역은 저장되며 관리자 페이지에서 열람 가능
- 삭제/숨김 콘텐츠는 복원 가능
- 카테고리·태그 등의 정책은 관리자 페이지에서 수정
- 콘텐츠 페이지에서 제재는 가능하지만, 해제는 관리자 페이지에서만 가능

---

### 키보드 정보 페이지
- 키보드 정보 열람
- 불량 한줄평 삭제

### Q&A 게시판
- 열람 (모든 유저 가능)
- 답변 작성 (작성자·관리자)

### 키보드 소식 게시판
- 글 등록, 수정, 삭제
- 공지사항 지정/해제
- 열람

### 익명 게시판
- 게시글 등록/열람
- 작성자 확인
- 공지사항 지정/해제
- 불량 게시글 삭제·수정
- 첨부파일 삭제
- 익명 닉네임 추가/삭제
- 이용자 제재

### 자유게시판
- 게시글 등록/열람
- 공지사항 지정/해제
- 불량 게시글 숨김·삭제·수정
- 첨부파일 삭제
- 이용자 제재

---

### 관리자 페이지 기능

#### 회원 정보 관리
- 제재 내역 확인/수정

#### 신고 처리
- 신고 내역 확인
- 제재 실행

#### 게시물 관리
- 삭제 내역 확인
- 게시글/첨부파일 복원

#### 키보드 용어 관리
- 용어 등록/수정/삭제
- 카테고리 등록/수정/삭제

#### 키보드 DB 관리
- 키보드 정보 등록/수정/삭제
- 카테고리 및 태그 관리
- 유저 태그 제안 승인 여부 결정

## 👤 회원 Action
(회원이 페이지별로 할 수 있는 기능 및 역할 정리)

### 키보드 용어
- 열람
- 검색

### 키보드 정보
- 검색
- 스크랩
- 태그 투표 입력/수정
- 태그 건의
- 별점 및 한줄평 입력

### Q&A 게시판
- 내 문의글 열람
- 문의글 작성

### 익명 게시판
- 게시글 열람/작성/수정/삭제
- 댓글 작성/수정/삭제
- 불량 게시글 신고

### 자유게시판
- 게시글 열람/작성/수정/삭제
- 댓글 작성/수정/삭제
- 불량 게시글 신고

### 마이페이지
- 내 정보 확인/수정
- 스크랩 키보드 확인
- 작성 글 내역 확인
- 별점 내역 확인
- 활동 포인트 확인
